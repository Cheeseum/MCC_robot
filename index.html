What files are we talking about?

  • Design files and schematics.
  • Arduino sketch files (cpp) to have the robot do things.
  • Arduino libraries for the stepper motors.
  • How-to & links for using the Arduino IDE.
  • A README on the project that will contain additional info on the robot project.

I wanna download the files.

Okay. Sounds good. Just navigate to the github page and click 'clone or download'.

I wanna modify the files.

Okay. Let's say you made some good changes. Just fork the repo on the github page. Then make your changes and submit a pull request. Then the pull request will be merged with the main repo.

Is using github necessary to contribute?

No. This is just an easy way to keep track of changes and manage files. Plus, some exposure to github and git is not a bad thing.
